# ContaCorrente




## Introdução

Uma __conta corrente__  é composta por diversos atributos essenciais para sua operação. 
Ela possui um __número de identificação único__ , um __saldo disponível__  e um __limite de débito__  estabelecido. 
Além disso, a conta mantém um __registro de movimentações financeiras__ , que englobam todas as transações realizadas.


## Funcionalidades

- __Saque__ Permite a retirada de valores, respeitando o limite máximo permitido.
- __Depósito__ Possibilita a adição de fundos à conta.
- __Consulta de saldo__: Fornece informações atualizadas sobre o montante disponível.
- __Emissão de extrato__: Lista todas as movimentações realizadas em um período específico.
- __Transferência entre contas__: Permite a movimentação de valores entre contas correntes distintas.
